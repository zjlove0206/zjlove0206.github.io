AdminLte后台UI框架

2017-05-07
  • 公共模板

AdminLTE – 非常流行的基于 Bootstrap 3.x 的免费的后台UI框架

框架推荐基于AdminLTE进行后台功能快速开发。并为此专门做一些规范改进。

  • 增加快捷部署的模块
  • 整合ionicons awesome 字体
  • 默认dist（字体、皮肤），整合以后目录为public/assets_default/admin-lte 对应的 css\img\js。详情查看 AdminLte下的Gruntfile.js
